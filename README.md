# lita-deploy

[Lita](https://www.lita.io/) handler for
[Github Flow](https://guides.github.com/overviews/flow/)

## Installation

Add lita-deploy to your Lita instance's Gemfile:

``` ruby
gem 'lita-deploy'
```

## Configuration

``` ruby
config.handlers.deploy.api_key = XYZ''
```

## Usage

lita deploy XYZ
